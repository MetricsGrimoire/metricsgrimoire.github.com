Welcome to Metrics Grimoire

Metrics Grimoire (usually pronounced in the French way) aims at providing a full set of tools to mine your favourite open source projects (included those hosted at GitHub!). This project was born in the Libresoft research group at University Rey Juan Carlos. After several years of work, a small community started to grow. Even later, a spin-off appeared, named Bitergia. In the meanwhile the old-named libresoft-tools were moved to a more neutral place where everyone is more than welcome to participate: Metrics Grimoire.

Channels of Communication

There are several ways where we will be more than pleased to help you:

  • IRC channel: #metrics-grimoire at freenode
  • Mailing list: metrics-grimoire at lists.libresoft.es
  • Source code adding patches (hosted at GitHub)
  • Bug Tracking Systems (hosted at GitHub).

The following is a list of the current set of tools with a brief description. If you are interested in more information, please visit each of the pages per tool or read the install or readme files.

CVSAnalY

Tool to retrieve information from the source code management system or versioning systems. This is supporting CVS, SVN and Git so far. It is in process the creation of an extension for Bazaar and Mercurial is in our minds.

Bicho

Tool to retrieve information from the bug tracking system.

MailingListStats

Tool to retrieve information from the mailing lists. Only supporting archives with mbox format.

Repository Handler

This is a library to handle repositories of source code

Cmetrics

This tool provides metrics about C
